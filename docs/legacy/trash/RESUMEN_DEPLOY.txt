PASO 1: Verificar requisitos
---------------------------------------------------------------------------
Comando:
  scripts\check-requirements.bat

Verifica:
  • Python ✓
  • Node.js ✓
  • npm ✓
  • Git (necesitas instalarlo si no lo tienes)
  • Docker (opcional)
  • Quartz clonado

Tiempo: 2 minutos

---------------------------------------------------------------------------

PASO 2: Ejecutar asistente de configuración
---------------------------------------------------------------------------
Comando:
  scripts\setup-complete.bat

Menú interactivo:
  1. Copiar contenido a Quartz
  2. Inicializar Git y conectar con GitHub
  3. Hacer primer commit y push
  4. Probar localmente (opcional)
  
Selecciona: 1 → 2 → 3 en orden

Tiempo: 10 minutos

---------------------------------------------------------------------------

PASO 3: Configurar GitHub Pages
---------------------------------------------------------------------------
En tu repositorio de GitHub:

  1. Ve a Settings
  2. Click en Pages (menú izquierdo)
  3. En "Source", selecciona: GitHub Actions
  4. Guardar

Tiempo: 2 minutos

Tu sitio estará disponible en:
  https://TU-USUARIO.github.io/sole-voltaje/
