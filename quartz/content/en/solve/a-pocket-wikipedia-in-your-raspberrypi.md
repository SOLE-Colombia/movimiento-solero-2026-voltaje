# **Install Kiwix on your Raspberry Pi**

## Welcome

Hi! I'm Gabriel, and I'm an artist. I love technology and I'm one of the first gadget makers at SOLE Colombia.

## What does it mean to install a pocket Wikipedia?

Wikipedia is a collaboratively built encyclopedia. Many people contribute their knowledge and expertise to this practically universal repository of information. Bringing all of Wikipedia's content to a device allows you to have offline access to a wealth of information with which you can learn many things.

If you share this information through an internal network with your Raspberry Pi, you can enable many people to learn with you too. That's what we want at SOLE Colombia: for people to discover how to take the Internet to the next level: as a group. Learn more about SOLE at [www.solecolombia.org](http://www.solecolombia.org/) to learn how communities are creating an equitable and beautiful future, together, one Big Question at a time.

## What's it for?

This recipe is part of a series of useful things you can do with your [**Raspberry Pi**](https://sole-doc.tiddlyhost.com/#Raspberry Pi). With a little work, you can set up a **small pocket internet connection** and turn it into a **portable wireless hotspot**. We can continue adding things to our Pi to make it a powerful mobile device, containing enough information to, among other things, use SOLE in places where there's no constant internet access. In the following steps, we'll put [Wikipedia](https://wikipedia.org/) on our Pi so we can consult it whenever we need to.

## What do you need?

- A **Raspberry Pi**
- A computer.

## How to do it?

- **Moment 1: Connect to your Pi**
    
    First, make sure you have a working Raspberry Pi. If you need help with this, check out the recipe for A Small Pocket Internet, where we show you how to get started with this device. Once you have your system up and running, connect to your Pi via a Terminal by starting an SSH connection. Make sure you have an internet connection to continue with the following steps.
    
- **Step 2: Download what you need**
    - The Kiwix server
        
        ![Imagen](/assets/images/Antena 3 4G 3 4G Antenna 22b2bd68c5b68076a24ccc4e1d6ff4af/image-800w.webp)
        
- 
- **Moment 2: Descarga lo necesario**
    - El servidor Kiwix
        
        The first component we are going to download is the application that will allow us to serve the compressed Wikipedia file, so that we can access it from our [local network](/lan-red-local-lan-local-area-network-conceptorio).
        
        ![Imagen](/assets/images/Conexión cable Ethernet Ethernet cable connection 22b2bd68c5b680fa90ceecc23bea76db/image 1-275w.webp)
        
        On the [Kiwix download page](https://www.kiwix.org/en/downloads/kiwix-serve/), copy the link titled GNU/Linux ARM (32-bit). Then, to download the file to your Pi, type the following into your terminal:
        
        ```
        wget https://download.kiwix.org/release/kiwix-tools/kiwix-tools_linux-armhf.tar.gz
        ```
        
    - The compressed Wikipedia
        
        ![Imagen](/assets/images/A pocket Wikipedia in your RaspberryPi 1a42bd68c5b680289571fc002238a69a/image 2-800w.webp)
        
        Now, let's download the compressed Wikipedia file. On the [Kiwix contents](https://library.kiwix.org/?lang=spa&category=wikipedia) page, find the version of Wikipedia that best suits your needs. We recommend downloading the 3.3GB version, which has enough information to do SOLE.
        
        Click the download button. Then copy the link to the *direct* icon and type this into your terminal:
        
        ```
        wget https://download.kiwix.org/zim/wikipedia/wikipedia_es_all_mini_2022-11.zim
        ```
        
        Please wait patiently for the process to complete.
        
- **Moment 3: Now, we install everything**
    - First the Kiwix server
        
        ![Imagen](/assets/images/A pocket Wikipedia in your RaspberryPi 1a42bd68c5b680289571fc002238a69a/image 3-800w.webp)
        
        Installing the Kiwix server is very simple; it only takes two steps:
        
        1. Unzip the file you just downloaded with the following command:
        
        ```
        tar -zxvf kiwix-tools_linux-armhf.tar.gz
        
        ```
        
        ![Imagen](/assets/images/A pocket Wikipedia in your RaspberryPi 1a42bd68c5b680289571fc002238a69a/image 4-800w.webp)
        
        1. We move the files to a system directory where the executable files are located:
            
            ```
            sudo mv kiwix-tools_linux-armhf-3.3.0-1/* /usr/local/bin
            ```
            
    - Now, the Wikipedia archive
        
        ![Imagen](/assets/images/A pocket Wikipedia in your RaspberryPi 1a42bd68c5b680289571fc002238a69a/image 5-800w.webp)
        
        Installing the .zim file involves moving it to another directory on your system so that the Kiwix server can easily find it:
        
        ```
        sudo mv wikipedia_es_all_mini_2022-11.zim  /var/local/
        ```
        
- **Moment 4: Let's create a service**
    
    ![Imagen](/assets/images/A pocket Wikipedia in your RaspberryPi 1a42bd68c5b680289571fc002238a69a/image 6-800w.webp)
    
    A service in Linux-like systems allows us to start, stop, and restart programs that run constantly on our operating system. Let's create a file with the service configuration so that the Kiwix server is automatically launched when the Pi boots.
    
    First we create the file:
    
    ```
    sudo nano /etc/systemd/system/kiwix.service
    ```
    
    and inside we write the following:
    
    ```
    [Unit]
    Description=Kiwix Service
    After=network.target
    
    [Service]
    Type=simple
    User=root
    Group=root
    ExecStart=/usr/local/bin/kiwix-serve /var/local/wikipedia_es_all_mini_2022-11.zim -p 8080
    Restart=always
    RestartSec=3
    
    [Install]
    WantedBy=multi-user.target
    ```
    
    We save the changes and we can continue with the recipe.
    
- **Step 5: We start our server**
    
    ![Imagen](/assets/images/A pocket Wikipedia in your RaspberryPi 1a42bd68c5b680289571fc002238a69a/image 7-800w.webp)
    
    The final step is to enable the new service and start it. With these two lines, the Kiwix server will start automatically every time we turn on our Pi.
    
    ```
    sudo systemctl enable kiwix
    sudo systemctl start kiwix
    ```
    
- **Moment 5: We test that everything works**
    
    ![Imagen](/assets/images/A pocket Wikipedia in your RaspberryPi 1a42bd68c5b680289571fc002238a69a/image 8-800w.webp)
    
    And now... the moment we've been waiting for!! Let's visit our pocket Wikipedia. In the browser of your computer or phone connected to the local network, navigate to the address [http://solevoltaje.local:8080](http://solevoltaje.local:8080/) . Once it loads, you can browse all the information we just downloaded :D
    

## How do you know if this solution works?

You've successfully filled your Pi with information! You now have a copy of the world's largest encyclopedia on a portable device. If you follow the recipes in this series, you can have a device that your community can connect to, search for answers to the Big Questions they have to solve, or so you can learn together.

## Why might this solution work?

You'll now be able to search your portable Wikipedia! But... it's a reduced version, and there are many other things you could find, if you have a stable internet connection. The great thing about this recipe is that you can have Wikipedia, take it with you to many places, and allow communities to connect to your Raspberry Pi!
