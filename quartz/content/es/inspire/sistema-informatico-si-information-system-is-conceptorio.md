# Sistema informático (SI) | Information system (IS)

idioma: english, español

## Español

Entendemos por **sistema informático** el **conjunto de elementos físicos y lógicos que se encargan de recibir, guardar y procesar datos para luego entregarlos en forma de resultados**. El esquema de un sistema informático engloba todo aquello que contiene una parte tangible (hardware) y otra lógica (software).

En la **actualidad**, los SI **ocupan un lugar clave para la organización humana**, ya que resulta ser una herramienta poderosa para el intercambio de información y la elaboración de redes informáticas que superan la barrera de las distancias.

### **Componentes de un sistema informático**

Los elementos que forman la **estructura básica** de todo sistema informático son:

1. **Hardware:** recoge todas las partes físicas del SI, es decir, todo lo que se puede tocar. En el caso de un ordenador, estos elementos del sistema informático serían: la pantalla, el teclado o el ratón.
2. **Software:** se refiere a la parte lógica de todo sistema informático, lo intangible. Es el conjunto de programas incluyendo datos, instrucciones, aplicaciones, etc, que permiten al usuario comunicarse con el hardware para poder realizar diversas tareas.
3. **Personal informático:** toda persona que utiliza de alguna manera el sistema, entre ellos, los creadores de software, los programadores o los usuarios en general.

### **Características de un sistema informático**

Algunas **características** que definen a un SI son:

- Los sistemas informáticos están **diseñados para procesar, almacenar, obtener y recibir información**.
- **Proporcionan una interfaz amigable** para su uso y gestión por parte de las personas.
- **Facilitan la automatización de los procesos**, ya que la programación de un sistema informático está dirigida a realizar tareas de forma autónoma.
- Su **objetivo** primordial es **ofrecer nuevas y mejores condiciones de vida** a los seres humanos.

### **Clasificación de los sistemas informáticos**

Los sistemas informáticos pueden clasificarse según su:

- **Volumen de procesamiento:** dependiendo de la capacidad que tienen para procesar y gestionar los datos.
- **Utilización:** es decir, el uso que se le da al sistema informático, ya sea para cumplir tareas con criterio reducido o para manejar un mayor número de variables.
- **Función:** según si es diseñado para cumplir propósitos puntuales dentro de una serie de trabajos. Podría estar programado para dar sugerencias en la toma de decisiones o para tomar una decisión propia a partir de datos recolectados.
- **Agrupamiento:** según sean sistemas aislados o formen parte de redes de equipos interconectados en redes privadas, locales, amplias, etcétera.

Una vez que comprendamos qué son los sistemas informáticos, pasamos a nombrar algunas **aplicaciones prácticas**, es decir, **para qué sirve un sistema informático en el mundo real**. (Fuente: Universidad Isabel I)

## English

A **computer system** is understood as the **set of physical and logical elements that are responsible for receiving, storing and processing data and then delivering them in the form of results**. The scheme of a computer system encompasses everything that contains a tangible part (hardware) and a logical part (software).

**Nowadays**, IS **occupy a key place in human organization**, as it is a powerful tool for the exchange of information and the development of computer networks that overcome the barrier of distances.

### **Components of a computer system**

The elements that form the **basic structure** of any computer system are:

1. **Hardware:** this includes all the physical parts of the IS, i.e. everything that can be touched. In the case of a computer, these elements of the computer system would be: the screen, the keyboard or the mouse.
2. **Software:** refers to the logical part of any computer system, the intangible part. It is the set of programs including data, instructions, applications, etc., that allow the user to communicate with the hardware in order to perform various tasks.
3. **Computer personnel:** any person who uses the system in some way, including software developers, programmers or users in general.

### **Features of a computer system**

Some **characteristics** that define an IS are:

- Computer systems are **designed to process, store, obtain and receive information**.
- **They provide a friendly interface** for its use and management by people.
- **They facilitate the automation of processes**, since the programming of a computer system is aimed at performing tasks autonomously.
- Their primary **objective** is to **offer new and better living conditions** to human beings.

### **Classification of computer systems**

Computer systems can be classified according to their:

- **Processing volume:** depending on the capacity they have to process and manage data.
- **Utilization:** that is, the use given to the computer system, either to fulfill tasks with reduced criteria or to handle a greater number of variables.
- **Function:** depending on whether it is designed to fulfill specific purposes within a series of jobs. It could be programmed to give suggestions for decision making or to make its own decision based on collected data.
- **Grouping:** depending on whether they are isolated systems or are part of networks of equipment interconnected in private, local, wide networks, etcetera.

Once we understand what computer systems are, we will go on to name some **practical applications**, i.e., **what a computer system is used for in the real world**. (Source: Isabel I University)
