## 🔐 Antes de empezar (solo si usas SSH)

Si ves error "Permission denied", necesitas SSH configurado:

```bash
# 1. Generar key
ssh-keygen -t ed25519 -C "tu-email@solecolombia.org"

# 2. Copiar key
cat ~/.ssh/id_ed25519.pub | clip

# 3. Agregar en GitHub
# https://github.com/settings/keys
```

O el script te ofrecerá cambiar a HTTPS automáticamente.
