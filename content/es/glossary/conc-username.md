Un nombre de usuario es una **cadena única de caracteres que identifica a un usuario en un sistema informático, como una red social, un sitio web o una aplicación**. 

Sirve para iniciar sesión y acceder a tu cuenta, y generalmente se usa junto con una contraseña para verificar tu identidad. (Fuente: [Dashlane](https://www.dashlane.com/es/blog/6-cosas-que-siempre-debe-tener-un-nombre-de-usuario-seguro))

**En detalle, un nombre de usuario:**

- **Identifica de forma única al usuario:** Permite distinguir a cada usuario dentro del sistema, evitando conflictos y garantizando que se otorguen los permisos correctos.
- **Se utiliza para iniciar sesión:** Es la forma en que te identificas para acceder a tu cuenta y a los recursos asociados a ella.
- **Puede ser público o privado:** En algunos sistemas, el nombre de usuario es visible para otros usuarios, mientras que en otros puede ser privado.
- **Debe ser único dentro del sistema:** No puede haber dos usuarios con el mismo nombre de usuario para evitar confusiones.
- **Puede ser alfanumérico:** Generalmente consiste en una combinación de letras y números, aunque algunos sistemas permiten caracteres especiales como guiones bajos o puntos.
- **Es diferente de la contraseña:** La contraseña es un factor de autenticación que protege el acceso a tu cuenta, mientras que el nombre de usuario es simplemente un identificador.
