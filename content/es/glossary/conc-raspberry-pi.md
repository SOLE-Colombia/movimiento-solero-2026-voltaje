### ¿Qué es?

Un [Raspberry Pi](https://www.raspberrypi.com/) es un computador del tamaño de una tarjeta de crédito que puedes configurar a tus necesidades.

Inicialmente, no cuenta con teclado, ratón o monitor, pero puedes conectarle estos y otros periféricos sin ningún problema.

Su "disco duro" es una memoria [**MicroSD**](/es/glossary/memoria-sd-minisd-microsd-sd-minisd-microsd-memory-conceptorio), en la cual se instala una versión de Linux, optimizada para este tipo de dispositivos, lo que te permite tener un computador muy portátil, que consume pocos recursos de electricidad, y en el que puedes hacer todo lo que normalmente haces con un computador de escritorio.

### ¿Cómo cargar tu Raspberry Pi?

Para cargar una Raspberry Pi, **necesitas conectarla a una fuente de alimentación.**

**La forma más común es a través de un cable USB a microUSB o USB-C, dependiendo del modelo, conectado a un adaptador de corriente USB.** Asegúrate de que el adaptador proporcione la tensión y corriente adecuadas para tu modelo de Raspberry Pi.

**Pasos:**

1. **Identifica el conector de alimentación:** Los modelos más antiguos, como la Raspberry Pi 3 y anteriores, utilizan un puerto microUSB, mientras que los modelos más nuevos, como la Raspberry Pi 4 y 5, utilizan USB-C. 
2. **Conecta el adaptador de corriente:** Conecta el adaptador de corriente USB al enchufe de pared y luego conecta el cable USB correspondiente al puerto de la Raspberry Pi. 
3. **Verifica la alimentación:** Si todo está conectado correctamente, la Raspberry Pi debería encenderse. Algunos modelos tienen luces indicadoras que te ayudarán a confirmar la alimentación.

### Ten en cuenta:

- **Voltaje y corriente:** La Raspberry Pi necesita una fuente de alimentación de 5V. La corriente requerida puede variar según el modelo, pero generalmente se recomienda entre 2.5A y 3A para modelos como la Raspberry Pi 4. (Fuente: [Bret](https://bret.dk/how-to-power-the-raspberry-pi-5-a-complete-guide/)).
- **Cargadores de teléfono:** Aunque algunos cargadores de teléfono pueden funcionar, es recomendable usar un adaptador de corriente diseñado específicamente para la Raspberry Pi para asegurar un rendimiento óptimo y evitar problemas de alimentación.
- **Hubs USB:** Si necesitas conectar varios dispositivos USB a la Raspberry Pi, considera usar un hub USB alimentado externamente para evitar problemas de suministro de energía.
- **Baterías:** Si necesitas alimentar la Raspberry Pi con una batería, asegúrate de usar un convertidor DC/DC para elevar el voltaje de la batería a 5V.
- **Raspberry Pi Pico:** La Raspberry Pi Pico puede alimentarse con una batería externa conectada a través de un cable microUSB a USB.

## English

### What is Raspberry Pi?

A [Raspberry Pi](https://www.raspberrypi.com/) is a computer the size of a credit card that you can configure to your needs.

Initially, it does not have a keyboard, mouse or monitor, but you can connect these and other peripherals without any problem.

Its "hard disk" is a [**MicroSD**](/memoria-sd-minisd-microsd-sd-minisd-microsd-memory-conceptorio) memory**,** in which a version of Linux is installed, optimized for this type of device, which allows you to have a very portable computer, which consumes few electricity resources, and in which you can do everything you normally do with a desktop computer.

### How to charge your Raspberry Pi?

To charge a Raspberry Pi, **you need to connect it to a power supply.**

**The most common way is through a USB to microUSB or USB-C cable, depending on the model, connected to a USB power adapter.** Make sure that the adapter provides the right voltage and current for your Raspberry Pi model.

**Steps:**

1. **Identify the power connector:** Older models, such as the Raspberry Pi 3 and earlier, use a microUSB port, while newer models, such as the Raspberry Pi 4 and 5, use USB-C. 
2. **Connect the power adapter:** Connect the USB power adapter to the wall socket and then connect the corresponding USB cable to the Raspberry Pi port. 
3. **Check the power:** If everything is connected correctly, the Raspberry Pi should turn on. Some models have indicator lights that will help you confirm power.

### Keep in mind:

- **Voltage and current:** The Raspberry Pi requires a 5V power supply. The current required can vary by model, but generally 2.5A to 3A is recommended for models such as the Raspberry Pi 4, [**according to bret.dk**](https://bret.dk/how-to-power-the-raspberry-pi-5-a-complete-guide/).
- **Phone chargers:** While some phone chargers may work, it is recommended to use a power adapter designed specifically for the Raspberry Pi to ensure optimal performance and avoid power issues.
- **USB hubs:** If you need to connect multiple USB devices to the Raspberry Pi, consider using an externally powered USB hub to avoid power supply issues.
- **Batteries:** If you need to power the Raspberry Pi with a battery, be sure to use a DC/DC converter to boost the battery voltage to 5V.
- **Raspberry Pi Pico:** The Raspberry Pi Pico can be powered by an external battery connected via a microUSB to USB cable.
