Una dirección IP es un **identificador único asignado a cada dispositivo conectado a una red que utiliza el protocolo de Internet**. 

En esencia, es como una dirección postal para tu dispositivo en el mundo digital, permitiendo que otros dispositivos te encuentren y se comuniquen contigo a través de Internet o una red local (Fuente: [Fortinet](https://www.fortinet.com/lat/resources/cyberglossary/what-is-ip-address#:~:text=Definición))

**Más detalladamente:**

- **Identificación única:** Cada dispositivo conectado a una red, ya sea una computadora, teléfono, impresora, etc., tiene una dirección IP única que lo identifica de manera inequívoca dentro de esa red.
- **Comunicación:** Las direcciones IP son fundamentales para que los dispositivos puedan enviar y recibir datos entre sí. Sin ellas, no podrían encontrarse ni intercambiar información.
- **Dos versiones:** Existen dos versiones principales de direcciones IP: IPv4 e IPv6. IPv4 es la versión más antigua y utiliza números de 32 bits, mientras que IPv6 es más reciente y utiliza números de 128 bits, lo que permite un número mucho mayor de direcciones posibles.
- **Tipos de direcciones:** Además de la versión, las direcciones IP pueden ser públicas (visibles en Internet) o privadas (utilizadas dentro de una red local). También pueden ser estáticas (permanentes) o dinámicas (cambian con el tiempo).
- **Funcionamiento:** Cuando visitas un sitio web, tu dispositivo envía una solicitud a través de tu proveedor de servicios de Internet (ISP) utilizando tu dirección IP. El ISP localiza el sitio web y te envía la información de vuelta, utilizando tu dirección IP como destino.

## English

An IP address is a **unique identifier assigned to each device connected to a network using Internet Protocol**. 

In essence, it is like a postal address for your device in the digital world, allowing other devices to find and communicate with you over the Internet or a local network (source [Fortinet](https://www.fortinet.com/lat/resources/cyberglossary/what-is-ip-address#:~:text=Definición y explicación de la dirección IP&text=Una definición de dirección IP,redes se comuniquen entre sí.)).

**In more detail:**

- **Unique identification:** Each device connected to a network, be it a computer, phone, printer, etc., has a unique IP address that uniquely identifies it within that network.
- **Communication:** IP addresses are essential for devices to be able to send and receive data to and from each other. Without them, they would not be able to find each other or exchange information.
- **Two versions:** There are two main versions of IP addresses: IPv4 and IPv6. IPv4 is the older version and uses 32-bit numbers, while IPv6 is newer and uses 128-bit numbers, allowing for a much larger number of possible addresses.
- **Types of addresses:** In addition to version, IP addresses can be public (visible on the Internet) or private (used within a local network). They can also be static (permanent) or dynamic (change over time).
- **How it works:** When you visit a website, your device sends a request through your Internet Service Provider (ISP) using your IP address. The ISP locates the website and sends the information back to you, using your IP address as the destination.
